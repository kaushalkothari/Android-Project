String Processing Example

Function Work of Function
strlen() Calculates the length of string
strlcpy() Copies a string to another string
strlcat() Concatenates(joins) two strings
strlcmp() Compares two string

Example

#include
int main(){
char name[30];
printf("Enter name: ");
gets(name); //Function to read string from user.
printf("Name: ");
puts(name); //Function to display string.
return 0;
}
Though, gets() and puts() function handle string, both these functions are defined in "stdio.h" header file.
